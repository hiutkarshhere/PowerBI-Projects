# PowerBI Projects 📊

A curated collection of Power BI dashboards and datasets demonstrating expertise in data visualization, analytics, and storytelling with real-world scenarios.

## 📂 Repository Contents
1. **HR Analytics**
   - File: `01_HR Analytics_PowerBI.pbix`
   - Dataset: `01_HR Analytics CSV.csv`
   - Focus: Employee attrition, performance, and HR KPIs.

2. **E-commerce Sales**
   - Files: `02_Ecommerce Sales_PowerBI.pbix`
   - Datasets: `02_Ecommerce Sales_Details_CSV.csv`, `02_Ecommerce Sales_Orders_CSV.csv`
   - Focus: Sales trends, customer segmentation, revenue analysis.

3. **Live Weather Reports (API-based)**
   - File: `03_API BASED_Live Wheather Reports_PowerBI.pbix`
   - Focus: Real-time weather data integration via APIs.

4. **Data Professional Survey**
   - Files: `04_Data Professional Survey_PowerBI.pbix`, `04_Data Professional Survey_CSV.xlsx`
   - Focus: Survey insights on tools, salaries, and career trends.

5. **Dashboards PDF**
   - File: `00_PowerBI Dashboards_PDF_file.pdf`
   - Contains exported visuals for quick preview.

---

## 🚀 Features
- Interactive dashboards with slicers and drill-throughs
- Real-world datasets (HR, E-commerce, Surveys, APIs)
- Clean, accessible visuals (color-blind–friendly design)
- Demonstrates advanced Power BI skills: DAX, relationships, measures

## 🛠️ Tools & Technologies
- Power BI Desktop
- DAX (Data Analysis Expressions)
- CSV/XLSX datasets
- API integration
